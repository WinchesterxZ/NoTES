- Authntication 
1- Paswword-Based Authntication (  الموقع مثلا يطلب منه يوزر وباسورد)
2- OTP( One Time Password)  ( دي خطوة زيادة بعد ما تدخل باسورد يأكد برضه انت مين ) 
وف الحالة دي بتكون 2FA 
3- Biomatric Authntication ( بيفتح ببصمة الوش او اليد ) 
4 - Multi-Factor Authntication ( وده بيجمع من اكتر من طريقة من الي فاتو مع بعض )
5- OAUTH ( الموقع بيسمحلك تدخل ب اكونت جيت هاب او جوجل او ابل وكده )
6 - Token-Based Authntication قليل

- Authorization
1- Role-Based Authorization ( ده بيكون في اكتر من صلاحية كل واحده له وظيفة معينه ممكن اليوزر العادي يمل view  بس والادمن له صلاحبات اعلى وكده )
2 - Object-Based Authorization ( حسب الاوبجكت ازاي يعني مثال منشور  فيسبوك انت عامله برايفت طبيعي محدش غيرك يشوفه او سامح لناس معينه بس وهك>ا )
3 - Attribute-Based Authorization ( حسب صفاتك مثلا الموقع سامح بس ل 20 سنة يخشو او موقع يفتح للمصريين بس ا وهك>ا )
4 - Time-Based Authorization ( مثلا موقع فاتتح انك تسجل بس بليل مثال واقعي لو ctf  فاتحه تلاقي يقولك التسجيل لمدة 3 ايام وهيقفل فاهم )
